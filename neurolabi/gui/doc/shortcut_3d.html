3D Window Shortcut

Neuron Node Selection
Ctrl/Cmd+ASelect all SWC nodes
STurn on selection mode
1: select downstream nodes
2: select upstream nodes
3: select connected nodes
4: inverse selection
5: select small trees
Shift+RPaint selection rectangle
S: select nodes in the rectangle
T: select trees in the rectangle

Neuron Editing (When nodes are selected)
Backspace/DeleteDelete selected objects
CConnect selected SWC nodes
NConnect to the nearest SWC nodes (only work for a single selected node)
BBreak SWC node connections
< Decrease radius
> Increase radius
Ctrl/Cmd+ZUndo
Ctrl/Cmd+Shift+ZRedo

Other
ZLocate selected SWC nodes in 2D view
=Zoom in
-Zoom out
All ArrowsRotate
[Shift] ArrowsMove
Ctrl/Cmd + SSave SWC
Ctrl/Cmd + PlusIncrease SWC Size Scale
Ctrl/Cmd + MinusDecrease SWC Size Scale
Ctrl/Cmd + MChange SWC Display Mode
Ctrl/Cmd + BToggle the mode of displaying neurons in the tile box only
