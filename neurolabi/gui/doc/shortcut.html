Shortcut Keys

Same for both 2D and 3D windows if not specified.

(Ctrl for PC and Cmd for Mac)

Node Selection
ShortcutFunctionNote
Ctrl/Cmd+ASelect all SWC nodes
HTurn on selection mode (show hints too)Use [Shift] to trigger a selection directly.
For example, [Shift+1] selects downstream nodes directly.
1select downstream nodes
2select upstream nodes
3select neighboring nodes
4select passing branches
5select connected nodes
6inverse selection
7select small trees
Shift+RPaint selection rectangle3D only
Hold Shift to append selection.
Sselect nodes in the rectangle
Tselect trees in the rectangle
Ctrl/Cmd+Tselect terminal branches in the rectangle

BBreak SWC node connections
IInsert between selected nodes
Q/< Decrease radius
E/> Increase radius

SpaceExtend node

Ctrl/Cmd+ETurn on erasing
