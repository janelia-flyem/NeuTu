neutu --command --compute_seed <input> [--position <int> <int> <int>] [--size <int> <int> <int>] -o output

Arguments:

  • --command: this argument must be specified to use any command line function of neutu
  • --compute_seed: run neutu as seed computation command command.
  • <input>: input DVID configuration file, which should be a json file (see schema below).
  • --position: the start point of the data block to compute, including three integers correspoindg to (x, y, z).
  • --size: the size of the data block to compute, including three integers corresponding to (width, height, depth).
  • -o: output file, which can be a SWC file when the file ends with the swc extension or a plain text file.
DVID configuration: 
{
  "title": "DVID Seeding Schema",
  "type": "object",
  "properties": {
    "address": {
      "description": "Address of the DVID server (without port)",
      "type": "string"
    },
    "port": {
      "description": "Port of the DVID server",
      "type": "int"
    },
    "uuid": {
      "description": "UUID of the DVID data",
      "type": "string"
    },
    "gray_scale": {
      "description": "Name of the grayscale data",
      "type": "string"
    }
  }
}
